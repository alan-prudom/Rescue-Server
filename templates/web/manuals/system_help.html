üíª Chromebook (Crostini) Support

Chromebooks require specialized handling for package manager locks and browser integration.

  • Bust Locks: If you see "apt locked" errors, run chromebook_fix.sh.
  • Automatic Handover: The master bootstrap detects penguin and automatically upgrades to the Intelligent Agent.
  • Browser Reports: Diagnostic results pop up in your Chrome browser automatically.

üìù Web Dashboard Features

  • Live Activity Feed: Monitor all connected PCs in real-time at /feed/.
  • Auto-indexing: All files added to the folders on the Mac appear instantly in the lists.
  • Instant Evidence: Use the text area to paste terminal output or notes. Click "Save Evidence" to create a timestamped .txt file on the Mac.
Pro Tip: Dashboard refresh rates use Exponential Backoff (10s to 150s). Move your mouse or click anywhere to reset the timer to 10s for active monitoring.

‚ö†Ô∏è VNC & Connection Troubleshooting

If you see a black screen or cannot connect:

  1. Xorg Check: Ensure you are logged into "GNOME on Xorg" rather than Wayland.
  2. Firewall: The bootstrap attempts to disable ufw and firewalld. Check sudo iptables -L.
  3. Agent Log: On the Chromebook, check client_activity.log in the audit_logs directory on the Mac.
